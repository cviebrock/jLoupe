jLoupe 1.2

By Chris Iufer chris@iufer.com

Requires jQuery 1.4+

Example 1: Scaled Thumbnail

an image scaled using the width or height attribute

<img src="largesize.jpg" width="256" class="jLoupe">
Example 2: Linked Thumbnail

a thumbnail image with a fullsize image set as the link href

<a href="largesize.jpg"><img src="thumbnail.jpg" class="jLoupe"></a>

Configurable Options

  • loupe: set the width and height of the loupe
  • margin: the margin around the viewable area inside of the loupe
  • cursorOffset: set the x and y distance from the cursor. (note: this will not work if the offset is a negative integer)
  • bgColor: the background color of the loupe
